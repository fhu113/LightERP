## 主数据管理

### 会计科目

系统采用SAP ERP风格的扁平化科目结构，无父子层级关系。

- **科目编码**：纯数字，如 1001, 1122
- **科目类型**：资产(ASSET)、负债(LIABILITY)、权益(EQUITY)、成本(COST)、收入(REVENUE)、费用(EXPENSE)

### 客户管理

- 添加客户基本信息（编码、名称、联系方式）
- 设置信用额度
- 查看应收账款余额
- 销售订单自动更新应收账款

### 供应商管理

- 添加供应商基本信息
- 查看应付账款余额
- 采购订单自动更新应付账款

### 物料管理

- 物料编码和名称
- 规格型号
- 单位
- 成本价（采购入库时更新）
- 销售价（可设置默认值）
- 当前库存数量
- **销售价自动更新**：销售订单成交后，自动更新物料的销售价为最新成交价

## OTC销售流程

### 流程图

```
销售订单 → 发货管理 → 销售开票 → 收款管理
   ↓           ↓           ↓          ↓
 创建/确认   创建/确认    创建/确认   创建/确认
```

### 1. 销售订单

**功能**：
- 创建销售订单，选择客户和物料
- 设置数量和单价
- 自动计算金额和税额（13%）

**状态流转**：
- DRAFT（草稿）→ CONFIRMED（已确认）
- CONFIRMED → CANCELLED（已取消）

**操作**：
- 新增订单 → 确认订单 → 创建发货单

### 2. 发货管理

**功能**：
- 根据销售订单创建发货单
- 选择要发货的订单和物料
- 确认发货后扣减库存

**状态流转**：
- DRAFT → CONFIRMED → COMPLETED
- 任何状态 → CANCELLED

**库存影响**：
- 确认发货时，扣减物料当前库存

### 3. 销售开票

**功能**：
- 根据销售订单或发货单创建发票
- 自动计算税额（13%）
- 确认开票后更新客户应收账款

**状态流转**：
- DRAFT → ISSUED（已开票）
- ISSUED → PAID（已收款）
- DRAFT → CANCELLED

**财务影响**：
- 确认开票时，增加客户应收账款

### 4. 收款管理

**功能**：
- 创建收款单，可关联销售发票
- 支持多种收款方式：现金、银行转账、支票、信用卡
- 确认收款后更新客户应收账款

**状态流转**：
- PENDING → PAID
- PENDING → CANCELLED

**财务影响**：
- 确认收款时，减少客户应收账款

## PTP采购流程

### 流程图

```
采购订单 → 采购收货 → 采购发票 → 付款管理
   ↓           ↓           ↓          ↓
 创建/确认   创建/确认    创建/确认   创建/确认
```

### 1. 采购订单

**功能**：
- 创建采购订单，选择供应商和物料
- 设置数量和单价
- 自动计算采购金额

**状态流转**：
- DRAFT → CONFIRMED
- CONFIRMED → CANCELLED

### 2. 采购收货

**功能**：
- 根据采购订单创建收货单
- 确认收货后增加库存
- 自动记录采购单价用于成本核算

**状态流转**：
- DRAFT → CONFIRMED → COMPLETED
- 任何状态 → CANCELLED

**库存影响**：
- 确认收货时，增加物料当前库存
- 采用移动加权平均法更新成本价

**公式**：
```
新成本价 = (原库存金额 + 新入库金额) / (原库存数量 + 新入库数量)
```

### 3. 采购发票

**功能**：
- 根据收货单创建采购发票
- 自动计算税额（13%）
- 确认开票后更新供应商应付账款

**状态流转**：
- DRAFT → ISSUED
- ISSUED → PAID
- DRAFT → CANCELLED

**财务影响**：
- 确认开票时，增加供应商应付账款

### 4. 付款管理

**功能**：
- 创建付款单，可关联采购发票
- 支持多种付款方式：现金、银行转账、支票、信用卡
- 确认付款后更新供应商应付账款

**状态流转**：
- PENDING → PAID
- PENDING → CANCELLED

**财务影响**：
- 确认付款时，减少供应商应付账款

## 功能说明

### 库存管理

- **成本核算**：移动加权平均法
- **库存更新时机**：
  - 采购收货确认 → 增加库存
  - 销售发货确认 → 扣减库存

### 财务自动更新

- 销售发票确认 → 增加客户应收账款
- 收款单确认 → 减少客户应收账款
- 采购发票确认 → 增加供应商应付账款
- 付款单确认 → 减少供应商应付账款

### 单据操作规则

| 操作 | 前提条件 | 影响 |
|------|----------|------|
| 确认销售订单 | 草稿状态 | 锁定价格，可创建发货单 |
| 确认发货单 | 草稿状态 | 扣减库存 |
| 确认销售发票 | 草稿状态 | 增加应收账款 |
| 确认收款单 | 待收款状态 | 减少应收账款 |
| 确认采购订单 | 草稿状态 | 锁定价格，可创建收货单 |
| 确认收货单 | 草稿状态 | 增加库存和成本 |
| 确认采购发票 | 草稿状态 | 增加应付账款 |
| 确认付款单 | 待付款状态 | 减少应付账款 |

### 数据验证

- 物料销售价自动更新为最新成交价
- 收货数量不能超过订单可收货数量
- 付款金额不能超过发票剩余金额
